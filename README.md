# MPI-gemm-multiply
```
C=alpha*A*B + beta * C
```

## Settings using docker:
```bash
source dockervars.sh
```

## Build:
```
mpicc gemm.c -o gemm
```

## Run:
```
mpirun -np 5 gemm
```
